# meteo
Utilisez arduino avec WIZNET5100 (ou autre module internet) pour accéder à la météo en temps réel et à la prévision du lendemain.


## Exemple écrit sur le moniteur série
```c++
+-----------------------------+
|            Ville            |
+--------------+--------------+
|    Today     |   Tomorrow   |
|     26°C     |  17°C/29°C   |
|  Ensoleillé  |  Ensoleillé  |
+--------------+--------------+
```
